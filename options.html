Survey Autofill Extension Setup

Currently supports surveys from fieldwork.com and userinterviews.com

Fill in any information that you'd like to be filled out automatically and hit submit. If you have a survey open, you will need to refresh the page for the changes to be reflected. Multiple answers can be put for a single field by using commas as a separator, in which case either the first value, or first matching value will be used.

When the survey asks for a text input, your direct answer will be autofilled. When a multiple choice answer is prompted, matching options will be automatically selected. Matching options are found using matching by whole word, providing multiple answers per field will help in increasing likelihood of a matching option.

All information is stored locally on your computer.
